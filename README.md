# BangkokChallenge
방콕 챌린지
### **개발 환경 및 기술**

- Kotlin
- MVP
- Kakao SDK

### **세부 구현**

- 로그인 - Kakao OAuth
- 타임라인 (SNS 게시물)
- 게시물 작성
    - 파일 업로드 (Multipart 이용)
- 게시물 상세화면
    - 댓글 - 수정, 삭제
    - 좋아요
- 내 정보 화면
    - 게시물 수
    - 내 게시물 좋아요 수
    - 내 게시물 리스트
- 좋아요 페이지
    - 내가 좋아요 누른 게시물 리스트 타임라인
